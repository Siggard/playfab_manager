# PlayFab Bundle Editor

Визуальный редактор для управления каталогом PlayFab — бандлами, сущностями и их связями.

## Возможности

- Загрузка JSON-каталога из PlayFab
- Drag & Drop управление сущностями в бандлах
- Фильтрация и поиск по типам и названиям
- Создание и редактирование сущностей
- Автосохранение в localStorage каждые 30 секунд
- Экспорт обратно в JSON для загрузки в PlayFab

## Запуск

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev
```

Откроется на http://localhost:5173

## Использование

1. **Загрузка данных**: Нажми "Load JSON" и выбери файл каталога из PlayFab (формат `{ CatalogVersion, Catalog: [...] }`)
2. **Работа с бандлами**: Перетаскивай сущности из пула в бандлы и между бандлами
3. **Редактирование**: Клик на сущность открывает редактор
4. **Создание**: Кнопка "+" для создания новых сущностей
5. **Экспорт**: "Export JSON" для скачивания обновлённого каталога

## Структура данных

Редактор работает с JSON-каталогом PlayFab:
```json
{
  "CatalogVersion": "Main",
  "Catalog": [
    {
      "ItemId": "p_1",
      "ItemClass": "player",
      "DisplayName": "Player Name",
      "Bundle": null,
      ...
    },
    {
      "ItemId": "bundle_1",
      "ItemClass": "starter_pack",
      "Bundle": {
        "BundledItems": ["p_1", "p_2"]
      },
      ...
    }
  ]
}
```

## Автосохранение

- Автоматическое сохранение в браузер каждые 30 секунд
- При открытии предложит восстановить несохранённые данные
- Кнопка ручного сохранения в header

## Tech Stack

- Vue 3 (Composition API)
- Vite
- vuedraggable (drag & drop)
- IndexedDB / localStorage (автосохранение)
